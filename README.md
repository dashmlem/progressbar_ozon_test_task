# Progress Bar Component

## Обзор

В данном проекте был реализован переиспользуемый React-компонент "Progress Bar" для отображения прогресса выполнения процессов в мобильных и web-приложениях. Компонент предоставляет визуальное представление прогресса в виде кругового индикатора с различными состояниями и настройками.

## Основные состояния компонента:

- **Normal** - базовое состояние с управляемой дугой прогресса
- **Animated** - состояние с анимацией вращения
- **Hidden** - скрытое состояние

## API

Компонент предоставляет глобальный объект `ProgressAPI` для программного управления:

### Методы:

#### `ProgressAPI.setValue(value)`
Устанавливает значение прогресса (0-100).

#### `ProgressAPI.setAnimation(enabled)`
Включает или выключает анимацию вращения.

#### `ProgressAPI.setHidden(hidden)`
Скрывает или показывает компонент.

#### `ProgressAPI.getState()`
Возвращает текущее состояние компонента.

## Адаптивность

Компонент адаптируется под различные размеры экранов:

- **Десктоп/Планшет**: горизонтальное расположение элементов
- **Мобильные устройства (<360px)**: вертикальное расположение элементов
